## 🔧 GitHub Actions (Automation)

```yaml
# .github/workflows/update-stats.yml
# Auto-updates problem count badge on every push

name: Update DSA Stats
on:
  push:
    branches: [main]
  schedule:
    - cron: '0 0 * * 0'  # Weekly update

jobs:
  count-problems:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Count solved problems
        run: |
          count=$(find Topics/ -name "*.java" -o -name "*.cpp" -o -name "*.py" | wc -l)
          echo "Problems found: $count"
      - name: Update README badge
        run: echo "Badge update logic here"
```
