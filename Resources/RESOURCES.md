# 📚 Curated DSA Resources

> Hand-picked resources that actually help you level up.

## 🎯 Problem Platforms

| Platform | Best For | Difficulty |
|----------|----------|------------|
| [LeetCode](https://leetcode.com) | Interview prep, patterns | All levels |
| [Codeforces](https://codeforces.com) | Competitive programming | Medium–Hard |
| [GeeksForGeeks](https://geeksforgeeks.org) | Theory + problems | Beginner–Med |
| [HackerRank](https://hackerrank.com) | Structured tracks | Beginner |
| [CSES Problem Set](https://cses.fi/problemset/) | CP foundation | Medium–Hard |
| [AtCoder](https://atcoder.jp) | Math-heavy CP | Medium–Hard |
